# moments sharing App
Instagram like app built on Ionic mobile apps development framework.

![preview](http://i.imgur.com/1ZfhPUv.gif)

#### Installation
```
npm install
ionic serve
```

#### Features:
* Home screen
* Search screen
* Messaging/new message screens
* Profile/profile edit & options screens
* Preview of stories
* Camera plugin included
